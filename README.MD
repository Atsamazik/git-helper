# Основы работы с Git.


## Работа с командной строкой.  
```
У приложений существует графический интерфей GUI и интерфейс командной строки CLI.<\br>Через CLI пользователь взаимодействует с приложением, которое называют `консолью`,<\br>`терминалом` или  `командной строкой`.
```
### Основные команды и их описание:  
* `pwd` - выводит рабочую директорию;  
* `cd` - изменяет директорию на указанную;  
* `ls` - показывает содержимое директории;
